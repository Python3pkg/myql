Yahoo-OAuth

The YOAuth module is not supported anymore. mYQL comes with Yahoo-OAuth. From now on, this will be the library to use to when you want to use OAuth. Nothing has really changed anyway, since yahoo-oauth is based on YOAuth.

You can read the full documentation here

>>> import myql
>>> from yahoo_oauth import OAuth1
>>> oauth = OAuth1(None, None, from_file='credentials.json')
>>> yql = myql.MYQL(format='xml',oauth=oauth)
>>> response = yql.getGUID('josue_brunel')
